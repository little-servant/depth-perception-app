# Depth Lab

지각심리학 과제를 위한 인터랙티브 3D 깊이 지각 실험 웹앱입니다.

## Tech Stack

- Next.js 16
- React 19
- React Three Fiber
- Tailwind CSS 4
- Framer Motion

## Features

- `/`: 프로젝트 소개와 진입 CTA를 제공하는 랜딩 페이지
- `/explore`: 7가지 깊이 단서를 켜고 끄며 예측-관찰-설명 흐름으로 학습하는 탐험 페이지
- `/test`: 15개 2AFC 트라이얼로 깊이 단서 의존도를 확인하는 테스트 페이지

## Local Development

```bash
npm install
npm run dev
```

기본 개발 서버 주소는 `http://localhost:3000`입니다.
